Test01.java

Run - Run Configuration - Arguments 를 통해
정수를 2개 전달하면 해당 정수의 사칙연산 결과가
나타나도록 하시오.


Test02.java
사용자로부터 10 이상의 정수를 입력 받아 난수(0 ~ 9)로 나눈 결과를 출력한다.
위 과정을 100번 반복하여 출력하며, 나눌 수 없는 경우에는 나눈 결과를 0으로 대신 출력한다.


Test03.java
나이를 입력 받아 출력하시오. 나이가 0~150 사이의 범위를 벗어나면 AgeException 클래스를 이용하여
예외를 처리하시오. 나이를 입력 받기 위해 getAge() 메소드를 이용하시오.


Test04.java
주민등록번호(하이픈 포함)를 입력 받아 출력하시오. 주민등록번호가 14글자가 아니면 
PerIDException 클래스를 이용하여 예외를 처리하시오. 주민등록번호를 입력 받기 위해
getPerID() 메소드를 이용하시오.


Test05.java
UpDown 게임
1 ~ 100 사이의 난수를 발생시켜 해당 값을 맞출때까지 UpDown 게임을 진행하시오.
 다음의 경우 예외를 발생시키시오.
   1) 사용자가 1 ~ 100 이외의 값을 입력한 경우
   2) 정수 대신 다른 데이터(예 : 실수, 문자 등)를 입력한 경우
잘못 입력된 경우도 포함하여 몇 번의 시도만에 맞췄는지 출력하시오.
